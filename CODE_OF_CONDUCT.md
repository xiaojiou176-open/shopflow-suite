# Code of Conduct

## Our Standard

Shopflow is a technical collaboration space. We expect contributors to be:

- respectful
- evidence-driven
- honest about uncertainty
- careful with security and privacy

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment or discriminatory language
- publishing private information without permission
- deliberately misleading claims about readiness, verification, or publication
- submitting malicious code, fake tests, or fake evidence

## Enforcement

Project maintainers may edit, reject, or remove contributions that violate this
code of conduct.

## Reporting

For conduct problems, contact the maintainer privately through GitHub rather
than escalating through a public issue thread.
