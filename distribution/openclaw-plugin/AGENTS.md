# AGENTS.md — Shopflow OpenClaw fallback shell

This plugin is intentionally tiny.

It exposes a read-only Shopflow packet surface for OpenClaw fallback
distribution.

Canonical source
- `https://github.com/xiaojiou176-open/shopflow-suite`

Knobs
- `SHOPFLOW_PACKET_ROOT` (optional): override the packet directory shown by the
  tiny CLI output

Must not claim
- canonical Shopflow repo status
- official OpenClaw listing already live
- official OpenClaw org approval already granted
