# Popup Override — Shopflow UIUX Endgame R3

## Purpose

Popup is a quick router and one-glance summary.

## Must Have

- one dominant next action
- one secondary route
- one proof cue
- one short claim-boundary cue

## Must Not Have

- long report blocks
- stacked equal-weight cards
- more than one disclosure layer
- sidepanel-style dense governance text above the fold

## Layout

- first screen should fit the full decision frame on a narrow extension viewport
- route blocks may stack, but only the primary route should dominate
- details belong in one compact disclosure drawer at most

## Review Check

- if popup needs reading before action, it failed
