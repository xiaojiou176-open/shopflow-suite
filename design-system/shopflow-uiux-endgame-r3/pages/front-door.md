# Front-Door Override — Shopflow UIUX Endgame R3

## Purpose

Front door copy must read product-first and builder-first, not robotic packet-first.

## Must Have

- what it is
- why it matters
- what still is not claimed
- shortest truthful entry path

## Must Not Have

- install slab as the first impression
- packet shelf language before product identity
- robotic capability dump

## Review Check

- if a public reader feels they opened a tools packet before they opened a product repo, the front door failed
