# Suite Override — Shopflow UIUX Endgame R3

## Purpose

Suite is an internal-alpha concierge / lobby, not a ledger.

## Must Have

- `Start here`
- strongest 1-2 routes above the fold
- compressed claim-readiness signal
- deep sections that remain reachable through disclosure and hash routes

## Must Not Have

- rollout map as first visual anchor
- giant board-first experience
- internal control-plane sludge on first screen

## Layout

- first screen = guide, not inventory
- dense sections may exist below, but they must not dominate before routing is clear

## Review Check

- if the first impression is “internal state board”, Suite failed
